# Before Redemption

Read this fully before redeeming your key. These are the most common issues new users run into — knowing this upfront will save you time.

---

## Do I need to run AntiFlag?

Only if you've previously been banned or used another unlocker on this PC. Run it **once**, before your very first launch — never again after that. Re-running it at any point will trigger a HWID change.

If this is your first time and you've never used any unlocker before, skip it entirely.

---

## How do I manage my scripts?

The unlocker loads every file inside `C:/WGG` simultaneously. Only keep one active script in the folder at a time. To disable a script without deleting it, add `BAK` to the end of the filename:

> `_phoenix.wgg` ← active  
> `_warden.wggBAK` ← disabled

See [Double Script Execution](troubleshooting.md#double-script-execution) for more detail.

---

## What do I do after redeeming my key?

1. Run the **System Checker** to confirm your PC is compatible.
2. Follow the [Install and Launch](getting-started.md#install-and-launch) guide to complete setup.

---

## How many HWID resets do I get?

| Key | Free Resets |
|-----|-------------|
| 30-day | 1 |
| 90-day | 3 |

Additional resets are **$20 each**.
